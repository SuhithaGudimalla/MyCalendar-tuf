# 📅 Paper Planner – Interactive Productivity Calendar

A modern, aesthetic **paper-style interactive calendar** built with **React** that combines productivity tracking, task planning, mood journaling, and seasonal visual experiences.

This project focuses on **frontend engineering, UI/UX design, state management, and responsive design** while maintaining a clean product-level user experience.

# ✨ Key Features

## 📆 Interactive Calendar System
- Monthly calendar navigation  
- Current date highlighting  
- Weekend styling  
- Smooth UI transitions  
- Fully responsive design  

---

## 📝 Daily Notes & Mood Diary

Users can click any date to:

- Add notes  
- Select mood emoji  
- Edit entries  
- Delete entries  

### Features:

- Mood tracking system  
- Daily journaling support  
- LocalStorage persistence  
- Clean modal interface  

---

## 📊 Task Range Management

Users can select multiple date ranges to create important tasks.

Each task includes:

- Task title  
- Date range  
- Priority level  
- Color tag  

### Capabilities:

- Multiple task ranges  
- Edit tasks  
- Delete tasks  
- Visual indicators on calendar  
- Task summary panel  

---

## 📌 Important Dates Tracker

Users can mark important individual dates such as:

- Birthdays  
- Interviews  
- Deadlines  
- Meetings  
- Events  

### Features:

- Color tagging  
- Description support  
- Calendar indicators  
- Easy deletion  

---

## 😊 Mood Diary System

Each day can be associated with a mood emoji.

Examples:

- Happy 😊  
- Focused 😎  
- Tired 😴  
- Excited 🤩  
- Productive 💪  

Helps track emotional productivity patterns.

---

## 💬 Monthly Motivational Quotes

Each month displays a motivational quote to encourage productivity.

Example:

> "Let progress bloom one day at a time."

---

## 🖼 Dynamic Hero Section

The hero section includes:

- Seasonal background images  
- Month & year display  
- Quote overlay  
- Image upload option  

Users can:

- Upload custom cover image  
- Personalize their planner  

---

## 🌦 Seasonal Visual Effects

Background visuals change based on season:

**Winter**
❄ Snow particles  

**Spring**
🌸 Flower particles  

**Autumn**
🍂 Falling leaves  

These effects enhance realism while remaining subtle.

---

## 🎉 Birthday Celebration System

Users can set their birthday and receive:

🎊 Confetti animation automatically on their birthday.

Adds a personal and delightful touch.

## 🌗 Theme Switching

Supports:

- Dark Mode  
- Light Mode  

### Features:

- Smooth theme transitions  
- Consistent color system  
- Readability optimized  

---

##  Productivity Dashboard

Side panel shows:

- Total tasks  
- Important tasks  
- High priority tasks

Helps visualize productivity at a glance.
